#
# Opis projekta

Ideja projekta je bila izraditi aplikaciju za prikaz vremensk prognoze. Login za aplikaciju omogućen je preko Google SignIn-a. Aplikacija omogućuje pretraživanje lokacija, prikaz šestodnevne prognoze za lokaciju i prikaz detaljne prognoze za određeni dan. Za prikaz podataka aplikacija se spaja na API koji sadrži podatke o lokacijama I vremenskim prognozama.
